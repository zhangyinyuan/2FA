## 🛠️ 技术栈总结

### 前端技术

| 技术 | 用途 | 特点 |
|---|---|---|
| HTML5 | 页面结构 | 语义化、现代标准 |
| CSS3 | 样式和布局 | 响应式、Flexbox、渐变 |
| JavaScript | 客户端逻辑 | 原生，无框架依赖 |
| Web Crypto API | TOTP算法 | 浏览器原生密码学 |
| Fetch API | 网络请求 | 现代异步通信 |

**关键特性**：
- ✅ TOTP在客户端计算（密钥不上传服务器）
- ✅ 实时30秒倒计时
- ✅ 响应式设计，支持移动设备
- ✅ 无外部框架，轻量级

**核心功能**：

## 🚀 立即开始

### 5分钟快速开始
```bash
# 1. 进入项目
cd c:/dev/workspace/nguone/2FA

# 2. 安装依赖
npm install

# 3. 启动应用
npm start

# 4. 打开浏览器
# http://localhost:3000
```

### 在Linux上部署
```bash
# 1. 上传项目
scp -r 2FA user@server:/opt/

# 2. 启动
cd /opt/2FA
docker-compose up -d

# 3. 访问
# http://server-ip:3000
```
